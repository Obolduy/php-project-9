## Описание
Веб-приложение для легкого SEO-анализа веб-страниц.

## Установка и развертывание
```bash
git clone https://github.com/Obolduy/php-project-9.git

cd php-project-9

make install
```

## Запуск тестов и линтера
```bash
# Run tests
make test

# Run linter
make lint

# Run tests with coverage
make test-coverage
```
